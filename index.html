Setup Guide
One-time setup — takes about 15 minutes
Step 1 — Create Google Sheet

Go to sheets.google.com → Create new spreadsheet → Name it: Solar Material Tracker
Create two sheets (tabs): PO_Master and Pallet_Log

PO_Master columns: PO_Number | PO_Quantity | Line_Code | Material_Type | PO_Status | Received_Qty | Added_On

Pallet_Log columns: Pallet_Number | PO_Number | Line_Code | Material_Type | Pallet_Qty | Remarks | Scanned_On

Step 2 — Add Google Apps Script

In your Google Sheet → Extensions → Apps Script → Delete existing code → Paste the Code.gs file content → Save (Ctrl+S)

Step 3 — Deploy as Web App

In Apps Script → Deploy → New Deployment
Type: Web App
Execute as: Me
Who has access: Anyone
Click Deploy → Copy the Web App URL

Step 4 — Connect This App

Open material_tracker.html in a text editor → Find const SCRIPT_URL = '' near the top of the script section → Paste your Web App URL between the quotes → Save

Or open browser Console (press F12 → Console tab) and type: testConnection()
Step 5 — Share with Engineers

Host this HTML file on GitHub Pages (free) or any web server → Share the link with all 20 engineers via WhatsApp / Teams.

Default admin password: admin123 (change it in the Code.gs script)
